API: Timestamp microservice

  • Instructions:
  • Pass a string as a parameter. API will check to see whether that string contains either a unix timestamp or a natural languange date
  • If it does, it returns both the Unix timestamp and the natural language form of that date.
  • If it does not contain a date or Unix timestamp, it returns null for those properties.

Example usage:

https://api-timestamp-microservice.glitch.me/September%2012,%202017 https://api-timestamp-microservice.glitch.me/1505174400

Example output:

{ "unix": 1450137600, "natural": "December 15, 2015" }
